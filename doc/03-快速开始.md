# 快速开始

## 环境要求

- JDK 21+
- Maven 3.6+

## 克隆项目

```bash
git clone <repository-url>
cd lion-gateway
```

## 编译项目

```bash
mvn clean package
```

## 运行项目

### 方式一：IDE 运行

直接运行主类：
```
com.zhaoguhong.lion.gateway.LionGatewayServerApplication
```

### 方式二：JAR 包运行

```bash
java -jar lion-gateway-server/target/lion-gateway-server-1.0-SNAPSHOT.jar
```

## 验证服务

服务启动后，监听 9999 端口。

### 测试 GET 请求

```bash
curl http://localhost:9999/test?param1=value1
```

### 测试 POST 请求

```bash
curl -X POST http://localhost:9999/test \
  -H "Content-Type: application/json" \
  -d '{"key":"value"}'
```

## 常见问题

### 端口被占用

修改 `NettyServer.java` 中的端口号：
```java
int port = 9999; // 修改为其他端口
```

### 虚拟线程未启用

检查 JDK 版本 >= 21，且 `application.yml` 中配置：
```yaml
spring:
  threads:
    virtual:
      enabled: true
```
