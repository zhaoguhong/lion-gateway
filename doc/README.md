# Lion Gateway 文档

Lion Gateway 是一个基于 JDK 21 虚拟线程和 Netty 的高性能 API 网关。

## 文档目录

- [项目介绍](./01-项目介绍.md) - 项目概述、特性、技术栈
- [架构设计](./02-架构设计.md) - 系统架构、核心模块、设计模式
- [快速开始](./03-快速开始.md) - 环境搭建、运行指南
- [插件开发](./04-插件开发.md) - 如何开发自定义插件
- [配置说明](./05-配置说明.md) - 配置文件详解

## 项目特点

- 高性能：基于 Netty 和虚拟线程，支持百万级并发
- 易扩展：插件化架构，易于添加新功能
- 规则灵活：支持多种匹配模式的规则引擎
- 代码简洁：使用虚拟线程，避免响应式编程复杂性
